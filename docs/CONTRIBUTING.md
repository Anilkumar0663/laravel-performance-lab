# Contributing

Thank you for your interest in Laravel Performance Lab.

## Ways to Contribute

- Add new benchmark scenarios
- Improve documentation
- Report bugs
- Suggest performance optimizations
- Submit pull requests

## Guidelines

- Follow PSR-12 coding standards.
- Keep benchmark code simple and reproducible.
- Include benchmark results when adding a new comparison.
- Update the documentation for any new benchmark.

Thank you for helping improve this project!
